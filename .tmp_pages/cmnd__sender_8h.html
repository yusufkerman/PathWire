Detailed Description

PathWire command frame builder and transmitter helper.

This file defines the cmnd_sender class, which is responsible for building PathWire command frames and pushing them into a TX ring buffer.

The class is designed for embedded systems and supports ISR-safe usage, assuming the underlying ring_buffer implementation is ISR-safe.

Frame format: {p:<path>:d:<data>}

  • No dynamic memory allocation
  • No blocking operations
  • All output is byte-wise pushed to a TX queue
Note
This component does NOT perform transmission itself. It only prepares and enqueues bytes for transmission.
